Privacy Policy

Last updated: March 2026

🔒 Your Data Stays With You

Lab Results Viewer is designed with a zero-storage architecture. Your health data is never stored, saved, logged, or retained in any way.

  • No server-side storage — uploaded files are parsed in memory and immediately discarded. Nothing is written to disk or any database.
  • No user accounts — there are no accounts, logins, or user profiles.
  • No cookies or tracking — this site does not use analytics, advertising cookies, or third-party tracking scripts.
  • No data transmission — your health data is never sent to any third party. All processing happens between your browser and this server during the upload request only.
  • No logs of health data — server logs may record basic request metadata (timestamps, IP addresses) for operational purposes, but never the content of your uploaded files or any health information.

💾 Local Browser Storage

The only data stored locally in your browser (via localStorage) is your demographic preference (sex and age selection) — used solely to apply the correct reference ranges. This data never leaves your browser and can be cleared at any time through your browser settings.

📋 What We Process

When you upload a CSV or XML file, the server:

  • Parses the file to extract test names, values, dates, units, and reference ranges
  • Returns the structured data as JSON to your browser
  • Immediately discards the uploaded file from memory

All visualisation, charting, pattern analysis, and reference range adjustments happen entirely in your browser after the initial parse.

🔌 ManageMyHealth-Labs Chrome Extension

The companion Chrome extension ManageMyHealth-Labs can be used to export your lab results directly from the ManageMyHealth patient portal into CSV or XML format for use with this viewer.

  • The extension runs entirely within your browser and communicates only with the ManageMyHealth website you are already logged into.
  • The extension does not transmit your data to any external server, API, or third party.
  • Exported files are saved to your local device — you choose when and where to upload them.
  • The extension does not collect analytics, usage data, or telemetry.
  • The extension requires only the minimum browser permissions necessary to read lab result data from the ManageMyHealth portal.

🛡️ Data Security

  • All connections to this site are encrypted via HTTPS/TLS.
  • No health data is persisted on the server at any point.
  • The application source code is open and available for inspection on GitHub.

👤 Children's Privacy

This service is not directed at children under 16. We do not knowingly collect or process data from children.

📝 Changes to This Policy
